# 📊 Data Analyzer & Transformer (Python)

A simple yet powerful Python-based console application that allows users to **input, analyze, filter, sort, and transform data** in both **1D and 2D formats**.

---

## 🚀 Features

✔️ Supports **1D and 2D data input**
✔️ Generates **data summary** (Total, Sum, Min, Max, Average)
✔️ Calculates **factorial of a number**
✔️ Filters data based on **threshold value**
✔️ Sorts data in **ascending/descending order**
✔️ Works with both **lists and matrices**

---

## 🛠️ Technologies Used

* Python 3.x
* Basic Data Structures (Lists)
* Control Statements (Loops, Conditions)
